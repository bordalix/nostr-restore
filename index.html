Nostr restore

Restore your nostr backup

Works with files from nostr-backup

Know more

This app will restore your events from a nostr-backup file to a big list of relays.

If you want to use your own list of relays:

  • Save site or clone repo
  • Edit js/relays.js
  • Open index.html with a browser
